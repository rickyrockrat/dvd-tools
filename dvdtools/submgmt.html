News for release 0.3

Added preliminary support for utf-8 and iso8859-1 files.
Also added subtitles conversion to png files. This is very useful for files that have <i> and <b> tags. Submgmt will generate a png file for each subtitle and generate a correct xml for spumux. This way you can mux much nicer subtitles in your DVD than just muxing text files. As a side effect you can choose the font and the color of the subtitles.

History


Handling subtitles files may be frustrating. More often than not, the subtitle file is not really synchonized with the movie.  Submgmt is my answer to this problem.
It has many neat features :
- shift subtitles
- expand subtitles
- change subtitle fps
- synchronize with a movie
- shift position subtitles (very useful to correct srt generated by avidemux)

It knows the following formats :
- subviewer (.srt)
- microdvd
- subrip
- sami
- spumux

As a side effect, it can convert a subtitle file to and from any of the above format.
